# 差旅标准

1. **住宿补贴（按城市等级）**：
    - 一线城市（北京、上海、广州、深圳）：800元/晚。
    - 新一线及二线城市：500元/晚。
    - 其他城市：350元/晚。
2. **交通工具**：
    - 飞行仅限经济舱，限单程不大于2000元。
    - 高铁限二等座（经理及以上级别可选一等座），限单程不大于1000元。

# 差旅预算评估

如差旅目的地为杭州，时长2天；

住宿：因为是二线城市，500元/晚，住宿申报预算是500 * 2 = 1000元；
交通：有限选择高铁，往返预算是2000元；

所以，差旅预算是：2000 + 2000 = 4000元。
